¿Qué es una base de datos?

es un conjunto organizado de información o datos estructurados, almacenados electrónicamente en un sistema informático, que permite gestionar, recuperar, actualizar y analizar datos de forma eficiente

¿Para qué sirve una base de datos?

sirve para almacenar, organizar, gestionar y recuperar grandes volúmenes de información de forma sistemática, segura y rápida.

¿Qué es un campo?

es la unidad básica de información, representada como una columna en una tabla que almacena un tipo de dato específico (ej. nombre, fecha, edad) para todos los registros.

¿Qué es un registro?

Un registro en una base de datos, también conocido como fila o tupla, es una unidad única de información estructurada dentro de una tabla que representa un objeto, entidad o ítem específico (como un cliente o producto). Se compone de múltiples campos (columnas) que contienen datos relacionados y no pueden duplicarse.

¿Qué es un dato

es la unidad mínima de información, representada por símbolos, números o letras, que describe un hecho, entidad o característica específica sin tener contexto por sí mismo

¿Qué es un diagrama entidad relacion?

es una herramienta visual que representa la estructura de una base de datos, mostrando entidades (objetos/conceptos), sus atributos (características) y cómo se interrelacionan entre sí
